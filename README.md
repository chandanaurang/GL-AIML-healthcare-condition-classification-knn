# Patient Condition Prediction Using Supervised Machine Learning

## Overview

This project applies supervised machine learning techniques to classify patient conditions using biomechanical measurements. Multiple classification algorithms were implemented and evaluated to predict patient health categories based on diagnostic attributes.

## Problem Statement

Healthcare datasets often contain biomechanical measurements that can be used to identify underlying patient conditions. This project explores classification techniques to automate patient condition prediction and support data-driven healthcare analysis.

## Technologies Used

- Python
- Pandas

- Seaborn
- Scikit-learn

## Dataset

The dataset contains patient records grouped into:

- Normal
- Type H
- Type S

Each record includes biomechanical measurements such as:

- Pelvic Incidence
- Pelvic Tilt
- Lumbar Lordosis Angle
- Sacral Slope
- Pelvic Radius
- Degree of Spondylolisthesis
